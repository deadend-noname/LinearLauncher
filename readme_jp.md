# LinearLauncher

このソフトは、いわゆるアプリケーションランチャーです。比較的動作が軽いと思います。

## 基本操作

起動するとアプリの一覧が表示されるので、好きなものを選んで起動してください。なお、最近起動したアプリ5つは一番上に表示されます。個数は設定で変えることもできます。

検索欄になにか入力すると、その文字列を含むものだけ表示されます。また、候補が一つになっている場合、入力欄でEnterキー（あれば）を押しても起動できます。

## 非表示アプリの設定

設定画面から、一覧に表示しないようにするアプリを設定できます。

設定画面の「非表示アプリ設定」を押すと、現在の設定が表示されます。アプリの一覧が表示されるので、非表示にしたいもののチェックを外してください。

## グループ機能

最下部には最初「Default」というタブがあると思いますが、右側の＋ボタンを押すとタブを増やすことができます。

増やしたタブを長押しすると、そのタブで表示されるアプリを設定、タブの順序を変更、タブの削除などができます。なお、タブは「あらかじめ指定したアプリだけ表示する」という機能ですので、複数のタブに同じアプリを入れることも可能です。

## そのほか

- ピンチイン・アウトでズーム（というか文字サイズの変更）ができます。
- 長押しするとアプリの詳細画面に飛べます。
